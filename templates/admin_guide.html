Admin Guide

  • Managing types - images
    To support the responsive design employed on the landing page, images used to illustrate types should be 960 x 640 in dimension.
  • Managing types - names
    Type names should be added all lower case and singular (not plural). The site will display the types as capitalized and plural when required.

    The software will ensure that duplicate names are not allowed however please do check existing type names for similarity before adding or editing.
  • Managing categories - names
    Category names should be added all lower case and singular (not plural). The site will display the categories as capitalized and plural when required

    The software will ensure that duplicate names are not allowed however please do check existing category names for similarity before adding or editing.
